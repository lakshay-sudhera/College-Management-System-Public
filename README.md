# 🎓 College Management System

A full-stack MERN application for managing:

## 👨‍💼 Admin
- Manage users
- Manage courses
- Upload academic calendar

## 👨‍🏫 Professor
- Create assignments & tests
- Evaluate submissions
- Mark attendance
- View analytics

## 🎓 Student
- Submit assignments & tests
- View grades
- Track attendance
- View progress (charts)

## 🛠 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Cloudinary

## ⚙️ Setup

### Backend
npm install
npm run dev

### Frontend
npm install
npm run dev

## 🔐 Environment Variables
Create `.env` file using `.env.example`
